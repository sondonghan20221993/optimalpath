상세 분석

🔵 Orbit-16: 완벽 균등

방위각: 0°, 22.5°, 45°, 67.5°, 90°, ..., 337.5°
모두 고유: 16/16 ✓
특징: 기하학적으로 완벽한 원형 배치
SfM 이점: 모든 표면이 다양한 각도에서 관찰 → 특징점 안정적

🔴 Greedy-16: 극단적 불균등

방위각: 0~180° 집중, 180~360° 공백
Max gap 101.2°: 한쪽은 25° 간격, 반대쪽은 100°+ 공백
근본 원인: Coverage 최적화로 인한 tie-breaking
→ Stage 1이 100% coverage 후 Stage 2에서 모든 선택이 동등
SfM 문제: 반대쪽 영역 가려짐 → 특징점 부족 → 복원 불안정

🟢 Hybrid-16: 최적 조합

구성: Orbit-8 (Stage 1) + Greedy-8 (Stage 2)
방위각: Orbit과 동일하게 완벽 균등
고유 위치: 16/16 ✓
장점: 균등성(Orbit) + 정보성(Greedy) 동시 달성
SfM 결과: 균등한 관점 + 충분한 정보 → 최고 품질 복원

🎓 최종 결론

방위각 균등성은 SfM 복원 품질을 직접 결정하는 핵심 요소

균등한 관점 분포 → 다양한 각도에서 특징점 관찰
→ 삼각측량 조건 개선 → 복원 오차 감소
