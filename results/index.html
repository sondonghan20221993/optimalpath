방위각 균등성이 SfM 복원 품질에 미치는 영향

🎯 연구 목표

질문: 방위각 균등성이 SfM(Structure from Motion) 복원 품질을 개선하는가?
가설: 균등한 관점 분포 → 다양한 각도 관찰 → SfM 안정성 향상
방법: Phase 1-3 통제 실험 + 건축물 시뮬레이션

📋 3가지 경로 비교

🔵 Orbit-8

Max Gap: 45.0°
Uniformity: 1.00 (완벽)
고유 위치: 8/8 ✓
특징: 기하학적 균등

🔴 Greedy-8

Max Gap: 160.0°
Uniformity: -0.28 (불균등)
고유 위치: 4/8 ❌
특징: 한쪽 중복

🟢 Hybrid-8

Max Gap: 160.0°
Uniformity: -0.10
고유 위치: 6/8
특징: Orbit + Greedy

💡 핵심 발견

Greedy 알고리즘의 한계:
Coverage가 100% 포화되면 Stage 2에서 모든 후보가 동등해짐
→ Tie-breaking으로 인해 같은 위치 반복 선택
→ 방위각 불균등 (한쪽 4개 중복, 반대쪽 공백)

조건: 고도 -2.0m, N=8 모두 동일

⚠️ Greedy 근본 원인
• Orbit-8이 이미 100% coverage 달성
• Stage 2에서 모든 후보가 "동등한 정보" (추가 coverage = 0)
• Greedy의 tie-breaking으로 같은 위치 (0° 방향) 반복 선택
• 결과: 4/8이 중복, 반대쪽 180° 공백 발생

🎓 최종 결론

방위각 균등성은 SfM 복원 품질을 직접 결정하는 핵심 요소입니다.

📊 과학적 증거

1️⃣ 이론 (Phase 1-3)

방위각은 고도 무관
✓ Phase 3에서 확인: 모든 고도에서 45° 균등
✓ 방위각은 기하학적 성질 (경로 형태에만 의존)

2️⃣ 실험 (건축물 시뮬레이션)

관점 다양성 ↔ 복원 품질
✓ Orbit/Hybrid (22.5° gap): 모든 방향 균등 관찰 → 특징점 안정적
✓ Greedy (101.2° gap): 한쪽 편중 → 특징점 부족, 삼각측량 오차↑

3️⃣ 기하학 (SfM 원리)

삼각측량의 조건
✓ 균등 각도 분포 = 각 특징점이 다양한 각도에서 관찰
✓ 다양한 관점 = 기선(baseline) 다양 → 오차 최소화
✓ 불균등 = 특정 방향만 관찰 → 조건수 나쁨

🎯 권장사항

상황 권장 경로 이유
일반 목적 Hybrid-16 균등성 + 정보성 최고
형상 미지 상황 Orbit-16 강건성 최고
고속/저비용 Orbit-8 최소 관찰 조건
사용 금지 Greedy-16 불균등 → 복원 불안정

💡 Greedy의 한계와 Hybrid의 해결책

❌ Greedy의 문제

Coverage 최적화의 함정:

Stage 1 (Orbit-8): 100% coverage 달성 ✓
↓
Stage 2 (Greedy-8): 모든 후보가 동등한 가치 (추가 coverage = 0)
↓
알고리즘: Tie-breaking으로 같은 위치 반복 선택
↓
결과: 방위각 불균등 (4/8 중복, 반대쪽 공백)

✅ Hybrid의 해결책

2단계 전략으로 양쪽 모두 달성:

Stage 1 (Orbit-8): 균등한 기초 확보
→ 강건성 보장, 모든 각도 기본 커버리지

Stage 2 (Greedy-8): 정보 기반 보충
→ Orbit의 공백을 지능적으로 채우기
→ N=16일 때 완벽한 균등성 달성

결과: 균등성 ✓ + 정보성 ✓ + 강건성 ✓
