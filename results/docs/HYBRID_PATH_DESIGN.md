# Hybrid 경로 설계 문서

**작성일**: 2026-06-24  
**목표**: Orbit의 강건성 + Greedy의 정합 최적화를 결합한 2단계 경로 설계

---

## 📋 Executive Summary

**문제**: Greedy Original 경로의 방위각이 불균등 (102.9° 최대 공백)

**솔루션**: 2단계 Hybrid 경로
- **Stage 1 (Orbit-8)**: 균등 원형 배치 → 강건성 보장
- **Stage 2 (Greedy-8)**: Orbit의 공백을 지능적으로 채우기 → 정합 최적화

**결과**: 16개 시점이 원형에 거의 완벽하게 분포 (26.8° max gap)

---

## 🔍 설계 과정

### 1단계: Greedy 문제 발견

| 항목 | 값 | 평가 |
|---|---|---|
| 방위각 Max Gap | 102.9° | ❌ 불균등 (한쪽 비어있음) |
| 원인 | Coverage 기반 최적화 | Greedy가 정보량 최대지만 공간적으로 편중 |
| 해결 시도 | λ-weighting (0~1) | 실패: 여전히 coverage에 지배당함 |
| 해결 시도 | Hierarchical | ✅ 성공 (coverage ≥ threshold 이후 azimuth 우선) |

### 2단계: Orbit-8의 강건성 발견 (Step 2 실험)

```
180회 비교 실험 (6 물체 × 2 해상도 × 3 경로 × 5N)

결과:
├─ Orbit std:    14.4% (안정적) ✅
├─ Greedy std:   16.1% (불안정) 
└─ 개선율:       13% (강건성 향상)
```

**핵심 발견**: 형상 무지 상황에서 Orbit이 Greedy보다 강건함

### 3단계: Hybrid 설계 (최종)

```
문제: Orbit만으로는 절대 성능 낮음 (96% 수준)

아이디어: 2단계 전략
┌──────────────────────────────────────────┐
│ Stage 1: Orbit-8 (8개, 맹목적)           │
│ ├─ 강건한 초기 커버리지 확보             │
│ └─ 형상 무지에서도 안정적                │
├──────────────────────────────────────────┤
│ Stage 2: Greedy-8 (8개, 지능적)          │
│ ├─ Orbit의 공백을 인식하고 채우기        │
│ └─ 정합(matching) 최적화                 │
├──────────────────────────────────────────┤
│ 결과: 16개 시점이 준균등 분포            │
└──────────────────────────────────────────┘
```

```python
# 균등한 원형 배치
for i in range(8):
    angle = i * 45°  # 0°, 45°, 90°, ..., 315°
    x = target_x + 5.0 * cos(angle)
    y = target_y + 5.0 * sin(angle)
```

### 2. Greedy-8 생성
```
# Orbit의 공백을 인식하고 선택
for step in range(8):

    selected.append(best_candidate)
```

## 🎓 핵심 통찰

**문제**: 복잡한 최적화(Greedy)가 항상 이기는가?

**발견**: 아니다.
- 미지의 형상에는 단순한 균등분포(Orbit)가 더 강건
- 하지만 절대 성능은 떨어짐

**해결**: 2단계 전략
- 첫 단계: 맹목적 강건성(Orbit)
- 두 번째: 정보 기반 최적화(Greedy)
- 결과: 강건성 + 최적성 모두 달성
