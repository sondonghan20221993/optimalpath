# 데이터가 말하는 진짜 이야기

## 📊 6개 그래프로 보는 정직한 분석

### 1️⃣ truth_1_per_object.png: 각 물체별 3경로 비교

**보여주는 것**: 각 물체에서 Orbit vs Greedy vs Random 누가 이기는가?

**결과**:
```
flat-simple:       100.0% = 100.0% = 100.0%  (회색: 동률)
flat-complex:      99.5% < 100.0% = 100.0%   (회색: Orbit만 약간 낮음)
box:               68.2% = 68.2% = 68.2%     (회색: 완전 동률)
sphere-small:      88.9% < 99.5% = 92.6%     (빨강: Greedy 우위)
sphere-large:      90.9% < 99.2% < 88.4%     (빨강: Greedy 우위)
occluded-chair:    63.2% = 63.2% = 63.2%     (회색: 완전 동률)
```

**해석**:
- **4개 물체 (흰색)**: 경로 선택이 무의미. 어떤 경로를 써도 같은 커버리지.
- **2개 물체 (sphere, 빨강)**: Greedy가 명백히 우위 (+10.6%, +8.3%).
- **Orbit 우위 물체**: 단 하나도 없음. 모두 동률이거나 Greedy에 밀림.

### 2️⃣ truth_2_std_vs_min.png: std vs 최솟값(worst-case) 비교

**문제**: 우리가 "std가 작음 = 강건"이라고 정의했는데, 정말 그럴까?

**왼쪽 그래프 (std)**:
```
Orbit std:   ~14.4%
Greedy std:  ~16.1%  ← Greedy가 더 큼
Random std:  ~14.5%
```

**오른쪽 그래프 (min/worst-case)**:
```
Orbit min:   63.2%
Greedy min:  63.2%   ← 완전 동일
Random min:  63.2%   ← 완전 동일
```

**깨달음**:
- Greedy의 std가 큰 이유 = sphere에서 혼자 높은 점수를 기록했기 때문
  - sphere-small: Orbit 88.9% → Greedy 99.5% (+10.6%)
  - sphere-large: Orbit 90.9% → Greedy 99.2% (+8.3%)
- 이건 "불안정"이 아니라 **"일부 물체에서 잘하기"**
- 반대로 Orbit의 std가 작은 이유 = 어디서나 고르게 비슷한 점수 (평범함)

**진짜 강건성 지표는 min**: 어떤 물체가 와도 최악이 얼마인가?
- Orbit min = 63.2% = Greedy min = Random min
- **Greedy도 63.2%에서 못 나간다** → Orbit의 강건성 주장이 무너짐

### 3️⃣ truth_3_gap_distribution.png: Greedy - Orbit 격차

**각 물체에서 Greedy가 Orbit을 몇 %p 이기는가?**

```
flat-simple:       0.0%   (동률)
flat-complex:      +0.5%  (미미)
box:               0.0%   (동률)
sphere-small:      +10.6% (Greedy 우위) ← 빨강
sphere-large:      +8.3%  (Greedy 우위) ← 빨강
occluded-chair:    0.0%   (동률)
```

**요약**:
- 회색(동률): 4개
- 빨강(Greedy 우위): 2개

**결론**: "Orbit이 강건하다"는 주장을 이 데이터로 지탱할 수 없습니다. 오히려 Greedy가 (구형 물체에서는) 명백히 우월합니다.

### 4️⃣ truth_4_distinguishability.png: 경로 구분 가능성

**이 물체에서 경로 선택이 의미가 있나?** → max(Orbit, Greedy, Random) - min의 범위

```
flat-simple:    ~37%  (경로에 따라 0~100% 가능)
flat-complex:   ~1%   (거의 동일)
box:            0%    (완전 동일 — 경로 무관) ← 빨강
sphere-small:   ~11%  (Greedy 선택 시 10% 이득)
sphere-large:   ~9%   (Greedy 선택 시 8% 이득)
occluded-chair: 0%    (완전 동일 — 경로 무관) ← 빨강
```

**치명적 발견**:
- **box (68.2%), occluded-chair (63.2%)**: 경로 구분 불가능
- 이 두 물체는 **천장(occlusion ceiling)에 막혀** 있음
- 어떤 경로를 쓰든, 심지어 Random도 같은 점수를 얻음
- **이 두 물체는 경로 비교 실험에 아무 정보도 기여하지 않습니다**

결론: 유효한 비교는 4개 물체뿐 (flat 2개, sphere 2개)

### 5️⃣ truth_5_sphere_greedy_dominates.png: 구형 물체에서 Greedy 우위

**sphere-small과 sphere-large에서 모든 N에 대해 Greedy > Orbit**

```
sphere-small (0.15m):
  N=4:  Orbit 87.8% < Greedy 97.7%
  N=6:  Orbit 89.4% < Greedy 98.9%
  N=8:  Orbit 88.9% < Greedy 99.5%
  N=12: Orbit 89.9% < Greedy 99.7%
  N=16: Orbit 90.5% < Greedy 99.8%

sphere-large (0.15m):
  N=4:  Orbit 89.8% < Greedy 99.1%
  N=6:  Orbit 90.6% < Greedy 99.4%
  N=8:  Orbit 90.9% < Greedy 99.2%
  N=12: Orbit 90.9% < Greedy 99.5%
  N=16: Orbit 90.9% < Greedy 99.6%
```

**그래프 특징**:
- Orbit 곡선: 거의 평탄 (N이 늘어도 ~90% 정도에서 포화)
- Greedy 곡선: 처음부터 높고 계속 상승 (95% 이상)
- Random 곡선: Orbit과 비슷 (또는 그 이하)

**해석**:
- **구형 물체는 정보가 풍부**: 어떤 시점에서도 큰 영역을 볼 수 있음
- **Greedy는 이 정보를 활용**: 유리한 시점을 골라서 높은 커버리지 달성
- **Orbit은 정보를 활용 못함**: 사전 정의된 원형이라 최적화 불가

### 6️⃣ truth_6_flatness_distribution.png: flatness 분포의 문제

**PCA flatness (작을수록 평탄)**:

```
box:             0.138  ← 극단 평탄
occluded-chair:  0.198  ← 극단 평탄
flat-simple:     0.167  ← 극단 평탄
flat-complex:    0.160  ← 극단 평탄

sphere-small:    0.933  ← 극단 구형
sphere-large:    0.972  ← 극단 구형
```

**목표 분포** (설계 문서):
```
0.25 ─ 0.35 ─ 0.45 ─ 0.55 ─ 0.65 ─ 0.75 ─ 0.85 ─ 0.92 (연속적)
```

**실제 분포**:
```
0.13~0.20 (5개 뭉침) ─────────────────────── 0.93~0.97 (2개 뭉침)
          극단 평탄                          극단 구형
```

**문제**:
1. **중간점 부재** (0.4~0.7): "형상 복잡도가 올라갈수록 포화점도 올라간다"는 단조성 입증 불가
2. **두 극단 군집**: "피어슨 r > 0.8" = 통계적 착시
   - 두 군집의 평균만 달라도 상관계수는 높아짐
   - 이건 상관이 아니라 "극단들의 분리"

**결론**: "형상 복잡도 vs 포화점" 상관 관계 무효

## 🎯 최종 진단: 데이터가 지원하는 것

### 거짓: "Orbit이 강건하다"

**근거**:
1. std(Orbit) < std(Greedy)는 "Orbit이 고르게 못함"을 의미하지, 강건함이 아님
2. min(worst-case)에서 모두 63.2% (Greedy도 같음)
3. 유효한 비교 물체는 sphere 2개뿐 (box/chair는 천장 차단)
4. 그 2개에서는 Greedy가 명백히 우위

### 참: "경로 선택의 의미는 제한적이다"

**근거**:
1. 6개 중 4개 물체: 경로 무관 (천장에 막힘)
2. 의미 있는 2개 (sphere)에서: Greedy > Orbit
3. Random도 box/chair에서는 Greedy와 동일 성능
4. 경로 효과가 나타나는 곳 = sphere의 정보 풍부한 영역뿐

### 참 (약함): "N=16은 모든 물체를 포화시킨다"

**근거**:
1. 포화점 분포: N=6 (평탄), N=8 (구형)
2. chair N=16: 0.00%p 증가분 (포화 맞음)
3. 하지만 이건 강건성이 아니라 **"상한 설정"** (N=16 이상은 효율 부족)

## 📋 정직한 결론

### 실험이 실제로 답한 질문

> **"미지의 물체를 복원할 때 균등 원형 경로(Orbit)가 형상 적응형(Greedy)보다 강건한가?"**

**답**: **아니다.** 오히려:
- 4개 물체에서는 경로 무관 (천장)
- 2개 물체(sphere)에서는 Greedy가 명백히 우위
- Orbit의 낮은 std는 강건함이 아니라 평범함

### 하지만 의미 있는 발견은 있다

1. **대부분의 물체는 경로 무관**: box(68.2%), chair(63.2%) 같은 가림 기하에서는 어떤 경로도 도움 못 함
2. **정보 풍부한 물체에선 Greedy 우위**: sphere처럼 모든 각도에서 보이면 Greedy가 최적화로 우위
3. **포화점은 실제로 N=6~8**: 16은 과도한 설정이며, 포화점만으로는 worst-case 보장 부족
